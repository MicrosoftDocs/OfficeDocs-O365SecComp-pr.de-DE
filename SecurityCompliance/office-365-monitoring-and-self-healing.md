<span data-ttu-id="0a3a9-120">Basierend auf der Kombination aus Fehlermeldung und roten Warnungen gibt diese Warnung genau an, welche Komponenten ein Problem aufweisen können, und dass das System versuchen wird, das Problem selbst zu beheben, indem ein Postfachserver neu gestartet wird.</span><span class="sxs-lookup"><span data-stu-id="0a3a9-120">Based on the combination of the failure alert and the Red Alerts, this alert indicates exactly which components could be having a problem, and that the system is going to try to correct the problem by itself by restarting a mailbox server.</span></span>
