## <a name="risk-awareness"></a><span data-ttu-id="0dbac-167">Risikobewusstsein</span><span class="sxs-lookup"><span data-stu-id="0dbac-167">Risk awareness</span></span>

<span data-ttu-id="0dbac-p115">Microsoft Secure Score ist eine numerische Zusammenfassung Ihrer Sicherheitslage basierend auf Systemkonfigurationen, Benutzerverhalten und anderen sicherheitsbezogenen Messungen. No online service is completely immune from security breaches, and secure score should not be interpreted as a guarantee against security breach in any manner.</span></span>
