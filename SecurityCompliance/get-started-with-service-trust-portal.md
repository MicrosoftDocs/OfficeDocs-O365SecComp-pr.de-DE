# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Erste Schritte mit Microsoft-Dienst als vertrauenswürdig-Portal

Das Microsoft Service vertrauen Portal (STP) bietet eine Vielzahl von Inhalten, Tools und andere Ressourcen zu Microsoft-Sicherheit, Datenschutz und Compliance-Methoden. Darüber hinaus unabhängigen Drittanbieter-Überwachungsberichte von Microsoft Onlinedienste und Informationen dazu, wie unseren online-Diensten können Ihre Organisation verwalten und Nachverfolgen von Compliance mit Standards, Gesetze und Bestimmungen, z. B.:
  
- International Organization for Standardization (ISO)
    
- Organisation steuert (SOC)
    
- National Institute of Standards and Technology (NIST)
    
- Federal Risiken und Autorisierung Management-Programm (FedRAMP)
    
- Allgemeine Data Protection Regulierung (GDPR)
  
## <a name="accessing-the-service-trust-portal"></a>Zugreifen auf die Vertrauensstellung Dienstportal

Das STP enthält ausführliche Informationen zur Implementierung von Microsoft Steuerelemente und Prozesse, die unsere Cloud-Dienste und die Kundendaten darin zu schützen. Zugriff auf einige STP Materialien melden Sie sich als authentifizierter Benutzer mit Ihrem Microsoft Cloud Services-Konto (einer Organisation Azure Active Directory-Konto oder eine Microsoft Account) und überprüfen und akzeptieren Sie den Microsoft nicht Offenlegung Lizenzvertrag für Compliance-Materialien.
  
 **Bestehende Kunden können das STP zugreifen** [https://aka.ms/STP](https://aka.ms/STP) mit einem der folgenden online Abonnements (Test- oder kostenpflichtige): 
  
- **Office 365**
    
- **Dynamics 365**
    
- **Azure**
    
> [!NOTE]
> Azure Active Directory-Konten, die Organisationen zugeordnet haben Zugriff auf den gesamten Bereich der Dokumente und Features wie Compliance Manager. Microsoft-Konten für die persönliche Verwendung erstellt haben Zugriff auf den Dienst vertrauen Portal Inhalt beschränkt. 
  
 **Neue Kunden und Auswerten von Microsoft online Services-Kunden**
  
Um ein neues Konto erstellen oder ein Testkonto erstellen, verwenden Sie eine der folgenden Anmeldung Formate (auch bei Testkonten verwendet), um Zugriff auf das STP abzurufen.
  
- Melden Sie sich für ein neues [Office 365 Business Testkonto](https://go.microsoft.com/fwlink/p/?LinkID=507653) oder ein neues [Testkonto Office 365 Enterprise.](https://go.microsoft.com/fwlink/p/?LinkID=698279)
    
- Melden Sie sich für ein neues [Testkonto Dynamics 365](https://go.microsoft.com/fwlink/?LinkId=252780)
    
- Melden Sie sich bei einem neuen [Azure Testkonto](https://go.microsoft.com/fwlink/?LinkId=722737). 
    
Wenn Sie sich für eine kostenlose Testversion oder ein Abonnement registrieren, müssen Sie Azure Active Directory zur Unterstützung des Zugriffs auf das STP aktivieren.

Der **Dienst vertrauen Portal** link verwendet, die Sie zur Homepage STP, einschließlich einer Neuigkeiten für das STP Abschnitt und Compliance-Manager, die Details über die aktuellsten Updates bereitstellt.

Verwenden Sie [Verwendung Compliance Manager zur Einhaltung von Datenschutz und behördlichen Anforderungen bei Verwendung der Microsoft-Cloud-Dienste](https://support.office.com/article/429e686f-d8a6-455e-a2b6-3791d763f000) zur Unterstützung der Einhaltung von Datenschutz- und gesetzlichen Anforderungen bei Verwendung von Microsoft Cloud Services. 
  
### <a name="trust-documents"></a>Vertrauensdokumente

Sichern Sie die Service vertrauen Portal ermöglicht den Zugriff auf die Vielzahl von Sicherheit und Informationen zum Entwurf mit dem Ziel für die Einhaltung von Vorschriften Ziele erfüllen, Sie verstehen, wie Microsoft Cloud Services die Daten behalten zu erleichtern. Um Inhalte zu überprüfen, wählen Sie eine Option aus dem Menü aus.
  
 - **Überwachen Reports** liefern unabhängige Prüfung und Bewertung von Berichten über die Microsoft Cloud Services-Kompatibilität mit Data Protection Richtlinien und gesetzliche Vorschriften, einschließlich: 
  
 - **Data Protection** bietet vertrauen Dokumente für den Download, Informationen zur Funktionsweise Microsoft Azure, Dynamics 365 und Office 365. 
  
 - **Azure-Sicherheit und Einhaltung von Bestimmungen Entwürfe** bietet einsatzbereiten Sicherheit und Compliance-Lösungen und unterstützen, auf die Anforderungen der Märkte, die die Cloud Annahme und Nutzung für Kunden mit regulierten oder eingeschränkte Daten beschleunigen zugeschnitten. 
  
### <a name="regional-compliance"></a>Regionale Compliance

Dieser Abschnitt enthält Regional bestimmte Kompatibilitätsdaten, häufig in Form von rechtliche Meinung, die Microsoft Cloud Services zu rendern.
  
 - **Tschechische Republik** enthält rechtliche Meinung Microsoft Onlinedienst Einhaltung Tschechische Republik Recht. 
  
 - **Polen** enthält rechtliche Meinung Microsoft online Service-Kompatibilität mit den Gesetzen des Polen. 
  
 - **Rumänien** enthält rechtliche Meinung Microsoft online Service-Kompatibilität mit den Gesetzen des Rumänien. 
  
 - **Spanien** enthält rechtliche Meinung über die Microsoft online-Dienst-Kompatibilität mit den Gesetzen des Spanien. 
  
### <a name="privacy"></a>Datenschutz

Diese Website enthält Informationen zu den Funktionen in Microsoft-Dienste, die Sie verwenden können, bestimmte Anforderungen von der GDPR Dokumentation hilfreich, um Ihre GDPR Accountability und Ihre Kenntnisse über die technischen und organisatorischen Maßnahmen beheben Microsoft hat zur Unterstützung der GDPR übernommen.
  
 - **GDPR: Erste Schritte** -Homepage für Inhalte Service vertrauen Portal GDPR im Zusammenhang mit Links zu relevante Inhalte und Tools zur Verfügung. 
  
 - **Daten Betreff Anforderungen** - wie Microsoft können Sie auf Daten Betreff Anforderungen, mit Links zu relevanten Dokumentation und Tools zu reagieren. 
  
 - **Daten Verletzung** - Informationen wie Microsoft erkennt und antwortet auf eine Verletzung der persönlichen Daten und benachrichtigt dann die Controllern unter GDPR, mit Links zu relevanten Dokumentation und Tools. 
  
 - **DPIA** - Informationen darüber, wie Microsoft Organisationen ihre eigenen Data Protection Auswirkungen Assessment Verpflichtungen erfüllen hilft. 
  
### <a name="resources"></a>Ressourcen

 - **Häufig gestellte Fragen** Antworten auf häufige und wichtige Fragen zu STP und Compliance-Manager. 
  
 - **Office 365-Sicherheit und Compliance Center** bietet umfassende Ressourcen für Informationen zur Sicherheit und Einhaltung von Vorschriften in Office 365, einschließlich Dokumentation, Artikel, und bewährten Methoden. 
  
### <a name="admin"></a>Administrator

Administrative Funktionen, die sind nur für das Administratorkonto eines Mandanten verfügbar und werden nur angezeigt, wenn Sie als globaler Administrator angemeldet sind.

Weitere Informationen finden Sie im Abschnitt "Berechtigungen und Role-based Access Control" in [Verwendung Compliance Manager zur Einhaltung von Datenschutz und behördlichen Anforderungen bei Verwendung der Microsoft-Cloud-Dienste](meet-data-protection-and-regulatory-reqs-using-microsoft-cloud.md#permissions-and-role-based-access-control).
  
### <a name="search"></a>Suche

![Service Vertrauensstellung Portal - Feld Suche Eingabe](media/7c5cd817-3d62-420b-adb4-76e33fef941f.png)
  
Klicken Sie auf das Lupensymbol in der oberen rechten Ecke der Seite vom Feld für die Suche zu erweitern, geben die Suchbegriffe aus, und drücken die EINGABETASTE. Das Search-Steuerelement angezeigt wird, mit den Suchbegriff in das Suchfeld input ein Bereich, und die Suchergebnisse unter angezeigt.
  
In der Standardeinstellung Suche gibt Dokument Ergebnisse zurück, und Sie die Dropdownlisten filtern nach zum Verfeinern der Liste der Dokumente angezeigt, zum Hinzufügen oder Entfernen von Suchergebnissen aus der Ansicht verwenden können. Mehrere FilterAttribute können zur selben Zeit Sie die zurückgegebenen Dokumente zu spezifischen Cloud Services, Kategorien von Compliance oder Sicherheitsmaßnahmen, Regionen der Welt oder Branchen einzuschränken. Klicken Sie auf den Link des Dokument zum Herunterladen des Dokuments.
  
![Service Vertrauensstellung Portal - Suche auf Dokumente mit angewendetem filter](media/86b754e1-c63c-4514-89ac-d014bf334140.png)
  
Klicken Sie auf den Link Compliance Manager, um die Anzeige von Suchergebnissen für Compliance Manager Assessment Steuerelemente. Die aufgeführten Suchergebnisse anzeigen dem Datum die Bewertung erstellt wurde, wird der Name der Bewertung gruppieren, die entsprechenden Cloud-Dienst, und gibt an, ob die Steuerelemente von Microsoft oder Kunden verwaltet werden.
  
![Dienst Vertrauensstellung Portal - Suche auf Compliance Manager-Steuerelemente](media/bafb811a-68ce-40b5-ad16-058498fe5439.png)
