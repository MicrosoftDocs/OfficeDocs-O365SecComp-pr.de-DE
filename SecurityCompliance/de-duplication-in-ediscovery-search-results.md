## <a name="how-duplicate-messages-are-identified"></a><span data-ttu-id="ab4d3-110">Identifizieren von doppelten Nachrichten</span><span class="sxs-lookup"><span data-stu-id="ab4d3-110">How duplicate messages are identified</span></span>

<span data-ttu-id="ab4d3-111">Office 365 eDiscovery-Tools verwenden Sie eine Kombination der folgenden e-Mail-Eigenschaften, um zu ermitteln, ob eine Nachricht doppelt vorhanden ist:</span><span class="sxs-lookup"><span data-stu-id="ab4d3-111">Office 365 eDiscovery tools use a combination of the following email properties to determine whether a message is a duplicate:</span></span>
  
- <span data-ttu-id="ab4d3-112">**InternetMessageId** – diese Eigenschaft gibt den Internet Nachrichtenbezeichner einer e-Mail-Nachricht an, bei der es sich um eine global eindeutige ID handelt, die auf eine bestimmte Version einer bestimmten Nachricht verweist.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-112">**InternetMessageId** - This property specifies the Internet message identifier of an email message, which is a globally unique identifier that refers to a specific version of a specific message.</span></span> <span data-ttu-id="ab4d3-113">Diese ID wird vom e-Mail-Clientprogramm des Absenders oder vom Host-e-Mail-System generiert, das die Nachricht sendet.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-113">This ID is generated by the sender's email client program or host email system that sends the message.</span></span> <span data-ttu-id="ab4d3-114">Wenn eine Person eine Nachricht an mehrere Empfänger sendet, ist die Internet Nachrichten-ID für jede Instanz der Nachricht identisch.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-114">If a person sends a message to more than one recipient, the Internet message ID will be the same for each instance of the message.</span></span> <span data-ttu-id="ab4d3-115">Bei nachfolgenden Überarbeitungen an die ursprüngliche Nachricht wird eine andere Nachrichtenkennung empfangen.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-115">Subsequent revisions to the original message will receive a different message identifier.</span></span> 
    
- <span data-ttu-id="ab4d3-116">**ConversationTopic** -his-Eigenschaft gibt den Betreff des Unterhaltungs Threads einer Nachricht an.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-116">**ConversationTopic** - his property specifies the subject of the conversation thread of a message.</span></span> <span data-ttu-id="ab4d3-117">Der Wert der **ConversationTopic** -Eigenschaft ist die Zeichenfolge, die das allgemeine Thema der Unterhaltung beschreibt.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-117">The value of the **ConversationTopic** property is the string that describes the overall topic of the conversation.</span></span> <span data-ttu-id="ab4d3-118">Eine Bestandserhaltung besteht aus einer anfänglichen Nachricht und allen Nachrichten, die als Antwort auf die ursprüngliche Nachricht gesendet wurden.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-118">A conservation consists of an initial message and all messages sent in reply to the initial message.</span></span> <span data-ttu-id="ab4d3-119">Nachrichten in derselben Unterhaltung haben denselben Wert für die **ConversationTopic** -Eigenschaft.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-119">Messages within the same conversation have the same value for the **ConversationTopic** property.</span></span> <span data-ttu-id="ab4d3-120">Der Wert dieser Eigenschaft ist normalerweise die Betreffzeile der anfänglichen Nachricht, die die Unterhaltung hervorgerufen hat.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-120">The value of this property is typically the Subject line from the initial message that spawned the conversation.</span></span> 
    
- <span data-ttu-id="ab4d3-121">**BodyTagInfo** -Dies ist eine interne Exchange-Informationsspeicher Eigenschaft.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-121">**BodyTagInfo** - This is an internal Exchange store property.</span></span> <span data-ttu-id="ab4d3-122">Der Wert dieser Eigenschaft wird durch Überprüfen verschiedener Attribute im Textkörper der Nachricht berechnet.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-122">The value of this property is calculated by checking various attributes in the body of the message.</span></span> <span data-ttu-id="ab4d3-123">Diese Eigenschaft wird verwendet, um Unterschiede im Nachrichtentext zu identifizieren.</span><span class="sxs-lookup"><span data-stu-id="ab4d3-123">This property is used to identify differences in the body of messages.</span></span>

<span data-ttu-id="ab4d3-130">Die folgenden Grafiken zeigen, wie doppelte Nachrichten in den Berichten **results. CSV** und **Manifest. <span data-ttu-id="ab4d3-145">Der Benutzer ändert dann eine der Kopien, indem er eine Anlage entfernt oder hinzufügt oder die Betreffzeile oder den Textkörper selbst ändert.</span><span
