## <a name="how-duplicate-messages-are-identified"></a><span data-ttu-id="66dd3-110">Identifizieren von doppelten Nachrichten</span><span class="sxs-lookup"><span data-stu-id="66dd3-110">How duplicate messages are identified</span></span>

- <span data-ttu-id="66dd3-112">**InternetMessageId** -diese Eigenschaft gibt die Internet Nachrichten-ID einer e-Mail-Nachricht an, bei der es sich um eine global eindeutige ID handelt, die auf eine bestimmte Version einer bestimmten Nachricht verweist.</span><span class="sxs-lookup"><span data-stu-id="66dd3-112">**InternetMessageId** - This property specifies the Internet message identifier of an email message, which is a globally unique identifier that refers to a specific version of a specific message.</span></span> <span data-ttu-id="66dd3-113">Diese ID wird vom e-Mail-Programm des Absenders oder vom Host-e-Mail-System generiert, das die Nachricht sendet.</span><span class="sxs-lookup"><span data-stu-id="66dd3-113">This ID is generated by the sender's email client program or host email system that sends the message.</span></span> <span data-ttu-id="66dd3-114">Wenn eine Person eine Nachricht an mehrere Empfänger sendet, ist die Internet Nachrichten-ID für jede Instanz der Nachricht identisch.</span><span class="sxs-lookup"><span data-stu-id="66dd3-114">If a person sends a message to more than one recipient, the Internet message ID will be the same for each instance of the message.</span></span> <span data-ttu-id="66dd3-115">Nachfolgende Überarbeitungen an der ursprünglichen Nachricht erhalten einen anderen Nachrichtenbezeichner.</span><span class="sxs-lookup"><span data-stu-id="66dd3-115">Subsequent revisions to the original message will receive a different message identifier.</span></span> 
    
- <span data-ttu-id="66dd3-116">**ConversationTopic** -seine Eigenschaft gibt den Betreff des Konversations Threads einer Nachricht an.</span><span class="sxs-lookup"><span data-stu-id="66dd3-116">**ConversationTopic** - his property specifies the subject of the conversation thread of a message.</span></span> <span data-ttu-id="66dd3-117">Der Wert der **ConversationTopic** -Eigenschaft ist die Zeichenfolge, die das allgemeine Thema der Unterhaltung beschreibt.</span><span class="sxs-lookup"><span data-stu-id="66dd3-117">The value of the **ConversationTopic** property is the string that describes the overall topic of the conversation.</span></span> <span data-ttu-id="66dd3-118">Eine Konservierung besteht aus einer anfänglichen Nachricht und allen Nachrichten, die als Antwort auf die ursprüngliche Nachricht gesendet wurden.</span><span class="sxs-lookup"><span data-stu-id="66dd3-118">A conservation consists of an initial message and all messages sent in reply to the initial message.</span></span> <span data-ttu-id="66dd3-119">Nachrichten innerhalb der gleichen Unterhaltung haben den gleichen Wert für die **ConversationTopic** -Eigenschaft.</span><span class="sxs-lookup"><span data-stu-id="66dd3-119">Messages within the same conversation have the same value for the **ConversationTopic** property.</span></span> <span data-ttu-id="66dd3-120">Der Wert dieser Eigenschaft ist in der Regel die Betreffzeile der ersten Nachricht, die die Unterhaltung hervorgebracht hat.</span><span class="sxs-lookup"><span data-stu-id="66dd3-120">The value of this property is typically the Subject line from the initial message that spawned the conversation.</span></span>
